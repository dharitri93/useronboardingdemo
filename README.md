# useronboardingdemo
A demo project for demonstrating the crud operation for different roles using Spring boot.
## Project structure 
Example
	|- src
	    |- main
	        |- java/com/dharitri
		|- controller
		    ApplicationController.java
		    Restcontroller.java
		|- model
		    User.java
		|- repository
		    UserRepository.java
		|- services
		    UserService.java
		Myapplication.java
	    |- resources
	        application.properties
	    |- webapp
		|- Web-INF/view
		    homepage.jsp
		    welcomepage.jsp
              pom.xml
###Note: Use your DB configuration to run the application.
